# To-Do List Web Application

A simple **To-Do List** application built using **HTML, CSS, and JavaScript**. This project implements **CRUD operations** (Create, Read, Update, Delete) for managing tasks efficiently.

---

## Features

- **Add Tasks**: Users can add new tasks to the list.
- **Delete Tasks**: Remove tasks from the list.
- **Mark as Completed**: Option to mark tasks as done.
